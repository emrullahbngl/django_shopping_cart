# emrullah-proje-shopping-cart-django-2.0-and-Python-3.6

**INSTALLATION**

``
pip install -r requirements.txt
``

**RUN SERVER**

``
python manage.py migrate --run-syncdb
``
``
python manage.py runserver
``

**ADD SUPER USER**

``
python manage.py createsuperuser
``
